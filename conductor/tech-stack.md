# Technology Stack

## Core
- **Programming Language:** Go (v1.25.5)
- **TUI Framework:** [Charmbracelet Bubble Tea](https://github.com/charmbracelet/bubbletea) - For the TUI lifecycle and state management.
- **TUI Components:** [Charmbracelet Bubbles](https://github.com/charmbracelet/bubbles) - For UI components like text inputs and tables.
- **TUI Styling:** [Charmbracelet Lip Gloss](https://github.com/charmbracelet/lipgloss) - For layout and terminal styling.

## Networking & Utilities
- **HTTP/HTML Parsing:** [golang.org/x/net](https://pkg.go.dev/golang.org/x/net) - For web crawling and HTML tokenization.
- **Headless Browser:** [Chromedp](https://github.com/chromedp/chromedp) - For crawling client-side rendered (SPA) websites.

## Infrastructure
- **Build System:** Makefile
- **CI/CD:** GitHub Actions (release workflow)
